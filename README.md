# logika része
ugyebár adott inputra én egy fát szeretnék megjeleníteni az input lebontásával,
tehát lenne maga az input mint gyökér, és akkor ágak stb...
na most a fa lebontásához és kiértékeléséhez lenne egy logikai rész, ami ezért felel,
ez értékeli ki az egyes gyerekeket, meg majd azt is hogy tautológia-e vagy sem, igaz/hamis halmazokat...
Illetve a felhasználó is tud majd egy másik fülön saját megoldást levezetni, és ezt is ki kell majd tudnia
értkelni.

# vizuális része
ezt a fát én vizuálisan is meg szeretném jeleníteni, így lenne egy olyan része, ami pl. előre
kiszámolja hogy milyen nagy lesz a fa, hány elem stb, és ezeket az adatokat vizuálisan elhelyezi, pl.
gyökér van legfelül, azt így középre helyezi, aztán megy lefelé és értelmesen megjeleníti az ágakat is
mígnem a gyerekekhez ér.

# alkalmazás része
ezt az egészet egy alkalmazás jeleníti meg, tehát lenne egy GUI oldala az egésznek. Én itt Kivyt tervezek használni jelenleg,
tehát lenne egy ablak, lenne egy input mező ahol a user megadja a kifejezést, és vizuálisan megjelenik az eredmény fa, illetve
másik fülön a user maga tud majd egy fát felépíteni.
